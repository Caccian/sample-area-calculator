## 使用流程

需准备激光测距仪和指南针：

1. **测高**：记录相机镜头到地面的垂直高度 `H`（若贴地测量填 0）。
2. **定位**：站在相机正下方。
3. **测量左侧 (B点)**：使用测距仪获取斜距 $r_B$，使用指南针获取方位角 $\phi_B$（正北为 0°，顺时针至 360°）。
4. **测量右侧 (C点)**：使用测距仪获取斜距 $r_C$，使用指南针获取方位角 $\phi_C$（正北为 0°，顺时针至 360°）。
5. **计算**：将数据填入本工具，计算样方的远端宽度、垂直纵深及**最终面积（公顷 / 平方米）**。

## 算法原理说明
1. **水平距离换算**:
   排除高度 $H$ 带来的干扰，将测距仪的斜距 $r$ 换算为地面水平距离 $d$：
   `d = √(r² - H²)`

2. **通过余弦定理推导远端宽度**:
   利用两点的方位角差值 $\Delta\phi$ 和水平距离，计算样方的远端宽度 $W_2$：
   `W₂ = √(d_B² + d_C² - 2·d_B·d_C·cos(Δφ))`

3. **通过向量叉积计算垂直纵深**:
   以站位为原点，建立平面直角坐标系。通过两点的平面坐标计算由相机和远端两点构成的三角形面积，进而求出该三角形底边（远端宽度）上的高，这就是样方的真实垂直纵深 $L$：
   `坐标: x = d·sin(φ), y = d·cos(φ)`
   `L = |x_B·y_C - x_C·y_B| / W₂`

4. **最终面积计算**:
   `S = W₂ × L`
