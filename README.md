## 使用流程

需准备激光测距仪和指南针：

1. **测高**：记录相机镜头到地面的垂直高度 `H`（若贴地测量填 0）。
2. **定位**：站在相机正下方。
3. **测左侧 (B点)**：测距仪打向样方远端左侧，记录斜距 `r_B` 与指南针方位角 `φ_B`。
4. **测右侧 (C点)**：打向远端右侧，记录斜距 `r_C` 与方位角 `φ_C`。
5. **计算**：将数据填入本工具，计算样方的远端宽度、垂直纵深及**最终面积（公顷 / 平方米）**。
